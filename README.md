## 🌟 الميزات الجوهرية (Core Features)

1. **🕵️ فحص ذكي (Advanced Threat Scanning):**
   - **الروابط:** ربط مباشر بـ Google Safe Browsing وتحليل محمي عبر VirusTotal.
   - **الملفات:** تحقق آمن بالاعتماد على البصمة الرقمية (SHA256 Hash) لمنع استنزاف الخوادم وتقليل أوقات الاستجابة بمساعدة نظام التخزين المؤقت (Caching).
   - **الصور و QR:** اكتشاف البكسلات المخفية وفك شفرات الـ QR محلياً لضمان أمن المحتوى.

2. **🤖 مساعد "Aman" الاستشاري (AI Sandbox):**
   - نظام سيبراني مزوّد بالذكاء الاصطناعي للإجابة على التهديدات والفحوصات، يتميز بإجابات قصيرة ودقيقة، ومحمي بشكل كلي ضد محاولات التلاعب بالمؤشرات النصية (Prompt Injection).

3. **🍯 المصيدة النشطة (Active Honeypot Network):**
   - خوارزمية ذكية لمراقبة المجموعات وكشف الإغراق البريدي، الروابط الخبيثة، وعمليات الاحتيال.
   - نقاط الخطر `Risk Scores` تحدد العقاب تلقائياً (تكميم عند 50 نقطة، حظر كامل وطرد عند 80 نقطة).
   - توفير فخاخ للمشرفين (Fake Commands) تحظر فورياً كل من يحاول استكشاف لوحة التحكم سراً.

4. **💳 نظام اشتراكات متكامل (Stripe Billing):**
   - دعم خطط اشتراك ذكية: `مجاني (5 فحوصات)`, `احترافي (100 فحص)`, `مؤسساتي (غير محدود)`.
   - معالجة مدفوعات تلقائية ومزامنة الباقات عبر **Stripe Webhooks** المؤمن بتقنية `HMAC`.

## 🏗️ البنية الهندسية (Under The Hood)

* **السرعة المفرطة**: يعمل ببيئة `Asyncio` كاملة غير مانعة (Non-Blocking).
* **إدارة الاتصالات المجمّعة (`_async_request`)**: دالة وحيدة لإدارة الاتصالات الخارجية تتولى معالجة الأخطاء وتنفيذ عمليات إعادة الاتصال (3 Retries).
* **الكفاءة (Caching)**: مدمج بذاكرة `LRU Cache` صالحة لمدة 60 دقيقة لكل فحص، لتصفير تكرارات واجهة تطبيقات VT.
* **رصد الأداء (Structured JSON Logging)**: السجلات تُبنى بشكل JSON لحفظ الـ `user_id` ومعرفات الأخطاء.
* **الأمان (Rate Limiting + Authorization)**: أزرار Inline Keyboards مرنة، وحد للطلبات لا يتجاوز 10 استعلامات في الدقيقة لمنع الإساءة (Spam).

## 🚀 دليل التثبيت والتشغيل (Deployment Setup)

### 1. إعداد البيئة المحلية
قم بنسخ المستودع عبر Git ثم هيئه:

```bash
# تثبيت الاعتماديات
pip install -r requirements.txt

# إنشاء الإعدادات وضبط الـ Tokens
cp .env.example .env
```

### 2. التكوين (Environment Variables)
في ملف `.env` يجب تحديد المتغيرات الأساسية للتشغيل كـ منتج حقيقي:
* `BOT_TOKEN` و `ADMIN_ID`
* `STRIPE_SECRET_KEY` و `STRIPE_WEBHOOK_SECRET`
* `VIRUSTOTAL_KEY` و `SAFE_BROWSING_KEY`
* `OPENAI_KEY`

### 3. التشغيل على أجهزة السحابة (Railway Deploy)
سيرفر مصغر مبني بمكتبة `aiohttp` سيبدأ بالعمل تلقائياً على البورت `8080`.
* يقوم بتقديم نقطة الـ Health Check على مسیر `/health`.
* يقوم بتقديم نقطة تأكيد الدفع التلقائي على مسير `/webhook/stripe`.

## 🖥️ أوامر النظام للمشرف (Admin Controls)
- `/admin`: لوحة التحكم الاحترافية (تظهر إحصائيات وإيرادات).
- `/hp_stats` و `/hp_recent`: جرد لآخر استهدافات وعمليات المصيدة ضد الحسابات.
